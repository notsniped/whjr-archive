# Project ADV-C154
This project code plots pixels of a watermelon on a pixel graph using the `matplotlib.pyplot` library.

## Requirements
* Python >=3.5
* `matplotlib` library

## Colour Map
Using `nipy-spectral` colour map (cmap).

### Color Values
* 4 - Dark Green
* 5 - Light Green
* 8 - Red
* 9 - Light Grey
